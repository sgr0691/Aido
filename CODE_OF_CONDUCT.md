# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and respectful environment for everyone.

## Our Standards

**Expected behavior:**
- Be respectful and constructive
- Accept feedback gracefully
- Focus on what's best for the project

**Unacceptable behavior:**
- Harassment, trolling, or personal attacks
- Publishing others' private information
- Other conduct that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject contributions that don't align with this Code of Conduct.
